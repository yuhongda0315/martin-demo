快速开始

快速开始

开始前请在 quick-start（ 项目目录 ）打开命令行窗口: Mac、Windows

1、安装 Node

Node 版本 (4.0.0+)

2、安装 gitbook-cli

npm install gitbook-cli -g

3、创建项目

创建 quick-start 目录或下载 示例

在 quick-start 目录中执行 gitbook init

4、预览

在 quick-start 目录中 gitbook serve

访问 http://localhost:4000
