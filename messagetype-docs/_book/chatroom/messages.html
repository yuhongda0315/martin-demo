消息类型

消息类型

发送渠道: 由 App Server 渠道发送的消息, SDK 只接收, 不可发送

名称 标识 描述 发送渠道 消息库版本
ChatroomStart RC:Chatroom:Start 开播通知 App Server 1.0.0
ChatroomEnd RC:Chatroom:End 结束通知 App Server 1.0.0
ChatroomSummary RC:Chatroom:Summary 信息汇总 App Server 1.0.0
ChatroomFollow RC:Chatroom:Follow 关注主播 SDK 1.0.0
ChatroomLike RC:Chatroom:Like 点赞 SDK 1.0.0
ChatroomGift RC:Chatroom:Gift 礼物 SDK 1.0.0
ChatroomBarrage RC:Chatroom:Barrage 弹幕 SDK 1.0.0
ChatroomWelcome RC:Chatroom:Welcome 欢迎用户进入聊天室 App Server 1.0.0
ChatroomAdminAdd RC:Chatroom:Admin:Add 设为管理员权限通知 App Server 1.0.0
ChatroomAdminRemove RC:Chatroom:Admin:Remove 移除管理员权限通知 App Server 1.0.0
ChatroomUserBlock RC:Chatroom:User:Block 用户封禁 App Server 1.0.0
ChatroomUserUnBlock RC:Chatroom:User:UnBlock 用户解除封禁 App Server 1.0.0
ChatroomUserBan RC:Chatroom:User:Ban 用户禁言 App Server 1.0.0
ChatroomUserUnBan RC:Chatroom:User:UnBan 解除用户禁言 App Server 1.0.0
ChatroomUserQuit RC:Chatroom:User:Quit 用户退出聊天室 SDK 1.0.0

ChatroomStart

聊天室创建完成后，主播可通过此消息在聊天室中发送具体的视频开播时间

属性 类型 描述 消息库版本
time string 开播时间，UNIX 时间戳 1.0.0
extra string 附加信息 1.0.0

示例

Android
iOS
Web
// 消息发送示例
ChatroomStart message = new ChatroomStart();
message.setTime("1525415951135");
message.setExtra("附加信息");
Message msg = Message.obtain("chrm2018", Conversation.ConversationType.CHATROOM, message);
RongIM.getInstance().sendMessage(msg, null, null, new IRongCallback.ISendMessageCallback() {
  @Override
  public void onAttached(Message message) {
  }
  
  @Override
  public void onSuccess(Message message) {
  }
  
  @Override
  public void onError(Message message, RongIMClient.ErrorCode errorCode) {
  }
});
// 消息发送示例
RCChatroomStart *message = [[RCChatroomStart alloc] init];
message.time = @"1525415951135";
message.extra = @"附加信息";

[[RCIM sharedRCIM] sendMessage:ConversationType_CHATROOM
                  targetId:@"chrm2018"
                   content:message
               pushContent:nil
                  pushData:nil
                   success:^(long messageId) {
                   } error:^(RCErrorCode nErrorCode, long messageId) {
}];
//消息发送示例
var ChatroomStart = RongIMClient.RegisterMessage.ChatroomStart;

var msg = new ChatroomStart({ 
  time: "1525415951135",
  extra: "附加信息"
});

var im = RongIMClient.getInstance();
var chatroomType = RongIMLib.ConversationType.CHATROOM;
var chatroomId = "chrm2018";
im.sendMessage(chatroomType, chatroomId, msg, {
  onSuccess: function(message) {
    console.log(message);
  },
  onError: function(error) {
    console.log(error);
  }
});
